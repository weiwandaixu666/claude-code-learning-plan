# Day 1 上午 - 1. 环境安装与配置

预计时间：30分钟

## 📚 学习目标

- [ ] 验证 Claude Code CLI 是否已安装
- [ ] 检查配置和账户状态
- [ ] 了解基本命令和帮助系统
- [ ] 完成第一次对话交互

## 🔧 第一步：验证安装

检查你的 Claude Code 版本：

```bash
claude --version
```

如果看到版本号，说明安装成功！

## ⚙️ 第二步：检查配置

查看当前配置：

```bash
claude config list
```

你应该看到：
- API 密钥状态
- 默认模型设置
- 其他配置选项

## 💬 第三步：了解帮助系统

Claude Code 有内置的帮助系统。尝试：

```bash
# 查看所有可用命令
claude --help

# 查看特定命令的帮助
claude commit --help
```

## 🎯 第四步：第一次对话

现在，让我们开始第一次对话！

**练习 1: 基础对话**

在 Claude Code 中输入以下内容：

```
你好！我是 Claude Code 的新用户。请简单介绍一下你自己能帮助我做什么？
```

**练习 2: 环境检查**

```
请帮我检查当前目录的结构，并告诉我这里有什么文件。
```

**练习 3: 获取帮助**

```
列出你最常用的 5 个工具，并简单说明它们的用途。
```

## 💡 常用快捷键和命令

```bash
# 退出当前会话
Ctrl + D 或输入 /exit

# 清除屏幕
Ctrl + L

# 暂停输出
Ctrl + S

# 恢复输出
Ctrl + Q
```

## ❓ 常见问题

**Q: Claude Code 没有响应怎么办？**
A: 检查网络连接，确认 API 密钥配置正确。

**Q: 如何查看所有可用命令？**
A: 输入 `/help` 或使用 `claude --help`

**Q: 我可以自定义配置吗？**
A: 可以！使用 `claude config set <key> <value>` 命令
